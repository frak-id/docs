# Getting Started with Wallet SDK

This guide will walk you through the process of setting up and using the Wallet SDK in your project. We'll cover product registration, SDK installation, and basic configuration.

## Prerequisites

Before you begin, ensure you have:

- Node.js (version 14 or later) installed
- A modern web browser
- Basic knowledge of JavaScript/TypeScript and your chosen framework (React or vanilla JS)

## 1. Register Your Product

:::info
Product registration is a crucial step that enables key functionalities of the Wallet SDK.
:::

Before using the Wallet SDK, you need to register your product on the Frak Business Platform:

1. Visit [https://business.frak.id/](https://business.frak.id/)
2. Sign up or log in to your account
3. Follow the product registration process
4. Set up your product details

For a detailed walkthrough, see our [Business Registration Guide](https://docs.frak.id/business/registration).

## 2. Install the Wallet SDK

Once your product is registered, install the Wallet SDK in your project:

:::code-group
```bash twoslash [npm]
npm install @frak-labs/nexus-sdk
```
```bash twoslash [yarn]
yarn add @frak-labs/nexus-sdk
```
```bash twoslash [pnpm]
pnpm add @frak-labs/nexus-sdk
```
```bash twoslash [bun]
bun add @frak-labs/nexus-sdk
```
:::

## 3. Basic Configuration

After installation, you need to configure the SDK. The configuration process varies slightly depending on whether you're using React or vanilla JavaScript.

:::code-group

```tsx twoslash [React]
// @noErrors
import { NexusConfigProvider, NexusIFrameClientProvider } from '@frak-labs/nexus-sdk/react'

const App = () => (
  <NexusConfigProvider 
    config={{
      walletUrl: 'https://wallet.frak.id',
      metadata: {
        name: 'Your App Name',
      }
    }}
  >
    <NexusIFrameClientProvider>
      {/* Your app components */}
    </NexusIFrameClientProvider>
  </NexusConfigProvider>
)
```

```ts twoslash [Vanilla JS]
// @noErrors
import { createIFrameNexusClient, createIframe } from '@frak-labs/nexus-sdk/core'

const client = await createIFrameNexusClient({
  config: {
    walletUrl: 'https://wallet.frak.id',
    metadata: {
      name: 'Your App Name',
    },
    domain: window.location.hostname,
  },
  iframe: createIframe({ walletBaseUrl: "'https://wallet.frak.id'" }),
})
```

:::

For more detailed configuration options, see our [Configuration Guide](/wallet-sdk/core-concepts/configuration).
