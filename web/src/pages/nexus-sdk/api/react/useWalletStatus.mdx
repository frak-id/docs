# useWalletStatus

## Summary
The `useWalletStatus` hook allows React components to subscribe to and monitor the current wallet status in real-time. It provides information about the connected wallet and any active interaction sessions.

## Requirements
- React environment
- Initialized `NexusIFrameClientProvider` in the component tree
- `@tanstack/react-query` library

## Hook Signature

```ts twoslash
// @noErrors
import { UseQueryResult } from '@tanstack/react-query';
import { WalletStatusReturnType } from '@frak-labs/nexus-sdk/core';

declare function useWalletStatus(): UseQueryResult<WalletStatusReturnType, Error>;
```

### Return Type
The hook returns a `UseQueryResult` object from `@tanstack/react-query` with the following structure:

```ts twoslash
// @noErrors
type WalletStatusReturnType = Readonly<WalletConnected | WalletNotConnected>;

type WalletConnected = {
  key: "connected";
  wallet: `0x${string}`;
  interactionSession?: {
    startTimestamp: number;
    endTimestamp: number;
  };
};

type WalletNotConnected = {
  key: "not-connected";
};
```
