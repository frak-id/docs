# watchWalletStatus

## Summary
The `watchWalletStatus` function sets up a listener to monitor changes in the Frak Wallet status, providing real-time updates on the wallet's connection state and interaction session.

## Requirements
- An initialized `NexusClient` instance

## Function Signature

```ts twoslash
// @noErrors
import type { NexusClient, WalletStatusReturnType } from '@frak-labs/nexus-sdk/actions';

declare function watchWalletStatus(
  client: NexusClient,
  callback: (status: WalletStatusReturnType) => void
): Promise<void>;
```

### Parameters
- `client`: `NexusClient` - An instance of the Frak client
- `callback`: `(status: WalletStatusReturnType) => void` - A function to handle wallet status updates

### Return Value
- `Promise<void>`: A promise that resolves when the listener is set up
