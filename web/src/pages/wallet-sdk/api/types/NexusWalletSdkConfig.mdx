# NexusWalletSdkConfig

The `NexusWalletSdkConfig` type defines the configuration options for the Frak Wallet SDK.

## Type Definition

```ts twoslash
type NexusWalletSdkConfig = Readonly<{
  walletUrl: string;
  metadata: {
    name: string;
    css?: string;
  };
  domain: string;
}>;
```

## Properties

### `walletUrl`
- Type: `string`
- The URL of the Frak Wallet service.

### `metadata`
- Type: `object`
- Metadata about your application.

#### `metadata.name`
- Type: `string`
- The name of your application.

#### `metadata.css`
- Type: `string | undefined`
- Optional CSS styles to customize the appearance of modals and SSO interfaces.

### `domain`
- Type: `string`
- The domain of your application. (Not required in react context)
