# useOpenSso

## Summary
The `useOpenSso` hook provides a convenient way to trigger the opening of the Single Sign-On (SSO) modal in the Nexus Wallet from a React component. This hook is useful for initiating the authentication process or allowing users to sign in to your application using the Nexus Wallet.

## Requirements
- Must be used within a component wrapped by `NexusIFrameClientProvider`
- Requires an initialized Nexus client

## Hook Signature

```ts twoslash
// @noErrors
import { UseMutationResult } from '@tanstack/react-query';
import { FrakRpcError, OpenSsoParamsType } from '@frak-labs/nexus-sdk/core';

declare function useOpenSso(options?: {
  mutations?: Omit<
    UseMutationOptions<void, FrakRpcError, OpenSsoParamsType>,
    "mutationFn" | "mutationKey"
  >;
}): UseMutationResult<void, FrakRpcError, OpenSsoParamsType, unknown>;
```

### Parameters
- `options` (optional): An object containing mutation options from React Query

### Return Value
- `UseMutationResult<void, FrakRpcError, OpenSsoParamsType, unknown>`: A mutation result object from React Query

## OpenSsoParamsType

The `OpenSsoParamsType` is defined as:

```ts twoslash
type OpenSsoParamsType = {
  redirectUrl?: string;
  directExit?: boolean;
  metadata: SsoMetadata;
};

type SsoMetadata = {
  logoUrl: string;
  homepageLink: string;
  links?: {
    confidentialityLink?: string;
    helpLink?: string;
    cguLink?: string;
  };
};
```

### Properties
- `redirectUrl` (optional): `string` - URL to redirect after SSO completion
- `directExit` (optional): `boolean` - Whether to exit directly after SSO
- `metadata`: `SsoMetadata` - Metadata for customizing the SSO modal
- `logoUrl`: `string` - URL of the logo to display
- `homepageLink`: `string` - Link to the homepage
- `links` (optional): Object containing additional link URLs

## Notes
- This hook uses React Query's mutation functionality, providing automatic retry and cache invalidation.
- The SSO process is handled entirely by the Nexus Wallet, simplifying the authentication flow in your application.
