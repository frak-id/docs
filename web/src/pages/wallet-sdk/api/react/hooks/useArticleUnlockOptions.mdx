# useArticleUnlockOptions

Hook used to get the unlock options for an article.

## Import

```tsx
import { useArticleUnlockOptions } from "@frak-labs/nexus-sdk/react";
```

## Usage

```tsx [index.tsx]
import { useArticleUnlockOptions } from "@frak-labs/nexus-sdk/react";
import type { Hex } from "viem";

function App({ article, contentId }: { article: { id: Hex; }, contentId: Hex }) {
  const { data: unlockOptions } = useArticleUnlockOptions({
    articleId: article.id,
    contentId
  });

  return (
    <div>
    </div>
  );
}
```

## Parameters

```ts
import type {
  GetUnlockOptionsParams
} from "@frak-labs/nexus-sdk/core";
```

### articleId

`Hex`

The article id to get the unlock options.

### contentId

`Hex`

The contentId to which this article belongs.

## Return type

```ts
import type {
  UnlockOptionsReturnType
} from "@frak-labs/nexus-sdk/core";
```

### TanStack Query `useQuery` return types.

[See all return types](https://tanstack.com/query/latest/docs/framework/react/reference/useQuery)

### data

`{ frkBalanceAsHex?: Hex, prices: { index: number; unlockDurationInSec: number; frkAmount: Hex; isUserAccessible: boolean | null; }[];`

The current user pFRK balance, and he prices for unlocking the article.
