# createIFrameNexusClient

## Summary
The `createIFrameNexusClient` function initializes a new Frak client instance, setting up the communication channel between your application and the Frak Wallet.

## Requirements
- A registered product on the Frak Business Platform
- A valid `NexusWalletSdkConfig` configuration
- An `HTMLIFrameElement` to host the Frak Wallet interface

## Function Signature

```ts twoslash
// @noErrors
import type { NexusWalletSdkConfig, NexusClient } from '@frak-labs/nexus-sdk/core';

declare function createIFrameNexusClient({
  config,
  iframe,
}: {
  config: NexusWalletSdkConfig;
  iframe: HTMLIFrameElement;
}): NexusClient;
```

### Parameters
- `config`: `NexusWalletSdkConfig` - Configuration object for the Wallet SDK
- `iframe`: `HTMLIFrameElement` - An iframe element to host the Frak Wallet interface

### Return Value
- `NexusClient`: An instance of the Frak client
