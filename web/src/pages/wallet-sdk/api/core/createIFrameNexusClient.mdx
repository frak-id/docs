# createIFrameNexusClient

## Summary
The `createIFrameNexusClient` function initializes a new Nexus client instance, setting up the communication channel between your application and the Nexus Wallet.

## Requirements
- A registered product on the Frak Business Platform
- A valid `NexusWalletSdkConfig` configuration
- An `HTMLIFrameElement` to host the Nexus Wallet interface

## Function Signature

```ts twoslash
// @noErrors
import type { NexusWalletSdkConfig, NexusClient } from '@frak-labs/nexus-sdk/core';

declare function createIFrameNexusClient({
  config,
  iframe,
}: {
  config: NexusWalletSdkConfig;
  iframe: HTMLIFrameElement;
}): NexusClient;
```

### Parameters
- `config`: `NexusWalletSdkConfig` - Configuration object for the Nexus SDK
- `iframe`: `HTMLIFrameElement` - An iframe element to host the Nexus Wallet interface

### Return Value
- `NexusClient`: An instance of the Nexus client
